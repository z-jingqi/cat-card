# 猫咪卡片游戏 - 设计文档

## 游戏概述

**猫咪卡片游戏**是一款回合制卡牌对战游戏，玩家通过使用猫咪卡片和辅助卡片来击败一系列BOSS。游戏强调策略性选择和资源管理，玩家需要在有限的回合内击败BOSS以通关。BOSS不会主动攻击玩家，游戏的胜利条件是将BOSS的血量降为零。

### 核心玩法
- 使用猫咪卡片攻击BOSS
- 使用辅助卡片增强战斗能力
- 在限定回合内将BOSS血量降为零
- 获得猫薄荷资源购买永久增益
- 逐步强化角色，挑战更高难度

## 卡片系统

### 猫咪卡片
猫咪卡片是主要的攻击手段，每张卡拥有基础攻击力和特殊能力。

#### 基本属性
- **攻击力**：对BOSS造成的基础伤害
- **特殊能力**：每种猫咪独有的技能效果

#### 猫咪种类
1. **缅因猫**
   - 攻击力：18
   - 特殊能力：**连击** - 攻击两次，每次造成50%伤害

2. **暹罗猫**
   - 攻击力：14
   - 特殊能力：**魔法飓风** - 对BOSS造成额外5点固定伤害

3. **孟加拉猫**
   - 攻击力：15
   - 特殊能力：**暴击** - 30%几率造成双倍伤害

4. **布偶猫**
   - 攻击力：12
   - 特殊能力：**群体增益** - 本回合其他猫咪卡片攻击力+3

5. **美国短毛猫**
   - 攻击力：16
   - 特殊能力：**穿透攻击** - 无视BOSS的任何抗性

### 辅助卡片
辅助卡片提供战术支持和增益效果，不直接造成伤害。

1. **猫薄荷精华**
   - 效果：本回合所有猫咪卡片伤害+5

2. **猫抓板**
   - 效果：本回合使用的第一张猫咪卡片效果触发两次

3. **逗猫棒**
   - 效果：获得额外的一次出牌机会

4. **猫铃铛**
   - 效果：从牌库中抽取2张猫咪卡片

5. **激光笔**
   - 效果：本回合所有猫咪卡片有20%几率造成双倍伤害

## 游戏机制

### 出牌限制系统
- 每场战斗有固定的出牌次数限制（如15次）
- 每回合可以出1-2张卡片（取决于游戏进度和增益）
- 回合结束时自动抽取新卡片补充手牌

### 回合机制
- 玩家每回合可使用指定数量的卡片
- 使用完当前回合的卡片后，回合自动结束
- 每场战斗的总出牌次数有限制

### 出牌顺序与效果触发
- 玩家可以在出牌前调整手牌的顺序（包括猫咪卡片和辅助卡片）
- 玩家可以调整永久增益的显示和触发顺序
- 辅助卡片的效果按从左到右的顺序依次触发
- 永久增益效果在辅助卡片效果后触发，也是从左到右顺序
- 猫咪卡片的攻击和特殊能力按从左到右顺序依次生效
- 调整卡片和增益顺序是战略的重要一环

### 卡片抽取系统
- **两个独立牌堆**：猫咪牌堆和辅助牌堆
- **初始手牌**：5张猫咪卡片和3张辅助卡片
- **重抽机制**：
  - 玩家可选择弃置不想要的卡片（最多4张，任意组合）
  - 从对应牌堆抽取等量替代卡片
  - 每小关卡限制重抽3次

### 牌库构成
- 猫咪牌库：约20张不同猫咪卡片
- 辅助牌库：约15张不同辅助卡片
- 游戏进程中可解锁更多卡片加入牌库

## 猫薄荷与商店系统

### 猫薄荷资源
- 每个小关卡结束后获得，数量基于：
  - 击败速度（回合数越少奖励越多）
  - 剩余生命值（生命值越高奖励越多）
  - BOSS难度（更难的BOSS给予更多基础奖励）
- 特殊成就可获得额外猫薄荷（如：不使用辅助卡完成关卡）

### 商店系统
- 每个小关卡结束后开放
- 提供3-5个商品槽位
- 可使用猫薄荷刷新商品选择
- 商品稀有度随游戏进度增加

### 永久增益
购买后在当前游戏冒险中持续生效，可以卖出获得部分猫薄荷返还。

#### 猫咪强化
1. **锋利爪牙** - 所有猫咪卡片攻击力+2/4/6
2. **灵敏反应** - 10/20/30%几率猫咪卡片造成额外伤害
3. **战斗天赋** - 每回合首次出牌造成15/25/35%额外伤害

#### 出牌系统增益
1. **额外行动** - 每回合可多出1张卡片
2. **连续出击** - 10%几率出牌后不消耗出牌次数
3. **战术调整** - 每场战斗可以额外重抽一次手牌

#### 卡片系统增益
1. **额外抽牌** - 每回合多抽1张卡
2. **精选手牌** - 每场战斗开始时可以选择更换1-2张初始手牌
3. **扩展容量** - 手牌上限+1/2

#### 战斗增益
1. **生命源泉** - 最大生命值+10/20/30
2. **战斗激励** - 连续使用同类型卡片时，效果提升10%
3. **爆发力量** - 每场战斗的第一张攻击卡牌伤害+20/30/40%

### 增益管理
- 初始可装备3个增益
- 游戏进程中可解锁更多栏位（最多6-8个）
- 增益可卸下、替换或卖出
- 某些增益组合在一起会产生协同效果

## BOSS系统

### BOSS特性
- 每个BOSS有不同的血量
- BOSS不会主动攻击玩家
- BOSS可以有被动技能（如：特定回合恢复少量生命值）
- 不同BOSS可能对某类卡片有抗性或弱点

### 示例BOSS
1. **训练假人** (教程BOSS)
   - 血量：100
   - 无特殊能力
   
2. **愤怒的狗狗**
   - 血量：150
   - 特性：对美国短毛猫卡片伤害减半

3. **机械老鼠**
   - 血量：200
   - 特性：每3回合恢复10点生命值

4. **黑暗猫头鹰**
   - 血量：250
   - 特性：对暹罗猫伤害免疫，但对孟加拉猫伤害加倍

5. **终极BOSS：喵星王**
   - 血量：300
   - 特性：每回合随机对一种猫咪品种免疫，玩家需要使用不同品种的猫咪来应对

## 游戏流程

### 单次冒险流程
1. 选择初始卡组和猫咪种类
2. 挑战第一个BOSS
3. 击败后获得猫薄荷
4. 访问商店购买增益
5. 挑战下一个BOSS
6. 重复直到通关或失败

### 游戏周期
- 单次通关为一个"冒险"
- 多次冒险累积解锁内容和成就
- 通过完成特定挑战解锁新的永久增益选项和卡片

## 界面设计

### 主要游戏界面
- 上方：BOSS区域，显示BOSS血量和状态
- 中间：战场区域，显示卡片效果和动画
- 下方：玩家区域，分为猫咪卡片和辅助卡片两个区域
- 左侧：玩家状态，显示生命值、能量和回合数
- 右侧：增益效果区，显示当前激活的永久增益

### 商店界面
- 展示可购买的增益和价格
- 显示当前猫薄荷数量
- 提供刷新和购买选项
- 增益详情和效果预览

## 未来扩展

### 潜在功能扩展
- 多样化的关卡环境，提供额外挑战
- 特殊事件系统，在关卡间出现随机事件
- 成就系统，记录玩家特殊战斗表现
- 解锁新的猫咪种类和辅助卡片
- 困难模式，提供额外挑战和奖励
